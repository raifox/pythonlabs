# 🐍 Python Laboratory Works (NLTU)

Цей репозиторій містить мої лабораторні роботи з програмування на мові Python, виконані під час навчання в НЛТУ.

## 📂 Структура репозиторію

Тут представлені скрипти, які демонструють різні навички: від базових математичних обчислень до аналізу даних.

- **Лабораторна робота 6 (`lab6_1.py`, `lab6_2.py`)**
  - Обчислення значень складних математичних функцій з обробкою винятків (`ZeroDivisionError`).
  - Знаходження кореня нелінійного рівняння методом половинного ділення (дихотомії) за допомогою рекурсії.

- **Лабораторна робота 7 (`lab7.py`)**
  - Робота з текстовими файлами (читання та запис).
  - Ручна фільтрація даних з `.csv` файлу (пошук за числовими діапазонами, точними значеннями та підстрічками).

- **Лабораторна робота 8 (`lab8.py`)**
  - Робота з масивами даних через бібліотеку `numpy`.
  - Зчитування датасету (`african_crises.csv`), багаторівневе сортування та запис результатів за допомогою вбудованого модуля `csv`.

- **Лабораторна робота 9 (`lab9.py`)**
  - Аналіз та обробка даних за допомогою бібліотеки `pandas`.
  - Фільтрація, сортування, групування (агрегація) та об'єднання даних (`merge`).

- **Лабораторна робота 10 (`lab10_1.py`, `lab10_2.py`, `lab10_3.py`)**
  - Візуалізація даних за допомогою бібліотеки `matplotlib`.
  - Побудова лінійних графіків, гістограм з анотаціями, використання підграфіків (`subplots`) та створення 3D-поверхонь.

- **Лабораторна робота 11 (`lab11.py`)**
  - Об'єктно-орієнтоване програмування (ООП).
  - Створення класів та успадкування, використання декораторів (`@property`, `@staticmethod`, кастомні декоратори), а також перевантаження магічних методів.

## 🚀 Як запустити

1. Склонуйте репозиторій на свій комп'ютер:
   ```bash
   git clone <посилання-на-ваш-репозиторій>
   ```
2. Переконайтеся, що у вас встановлено **Python 3**.
3. Для запуску останніх лабораторних робіт потрібно встановити додаткові залежності:
   ```bash
   pip install pandas numpy matplotlib
   ```
4. Запустіть потрібний скрипт з терміналу, наприклад:
   ```bash
   python lab8.py
   ```
