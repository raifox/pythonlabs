# 🐍 Python Laboratory Works (NLTU)

Цей репозиторій містить мої лабораторні роботи з програмування на мові Python, виконані під час навчання в НЛТУ.

## 📂 Структура репозиторію

Тут представлені скрипти, які демонструють різні навички: від базових математичних обчислень до аналізу даних.

- **Лабораторна робота 6 (`lab6_1.py`, `lab6_2.py`)**
  - Обчислення значень складних математичних функцій з обробкою винятків (`ZeroDivisionError`).
  - Знаходження кореня нелінійного рівняння методом половинного ділення (дихотомії) за допомогою рекурсії.

- **Лабораторна робота 7 (`lab7.py`)**
  - Робота з текстовими файлами (читання та запис).
  - Ручна фільтрація даних з `.csv` файлу (пошук за числовими діапазонами, точними значеннями та підстрічками).

- **Лабораторна робота 8 (`lab8.py`)**
  - Аналіз та обробка даних за допомогою бібліотек `pandas` та `numpy`.
  - Зчитування датасету (`african_crises.csv`), фільтрація, багаторівневе сортування, групування (агрегація) та об'єднання даних (`merge`).

## 🚀 Як запустити

1. Склонуйте репозиторій на свій комп'ютер:
   ```bash
   git clone <посилання-на-ваш-репозиторій>
   ```
2. Переконайтеся, що у вас встановлено **Python 3**.
3. Для запуску лабораторної роботи 8 потрібно встановити додаткові залежності:
   ```bash
   pip install pandas numpy
   ```
4. Запустіть потрібний скрипт з терміналу, наприклад:
   ```bash
   python lab8.py
   ```
